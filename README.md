# OSN-K Informatika Algorithm Review

Website statis untuk review dan analisis bagian algoritmika dari soal OSN-K Informatika Indonesia tahun 2021–2025.

## Fokus Proyek

Proyek ini tidak mencakup seluruh soal OSN-K, melainkan hanya bagian yang relevan dengan:
- source code C++ pada soal
- algoritma yang digunakan atau mirip
- struktur data yang relevan
- time complexity
- solusi ringkas
- latihan competitive programming yang serupa

## Fitur

- Daftar soal OSN-K Informatika per tahun
- Analisis source code C++ pada soal algoritmika
- Nama algoritma yang relevan
- Bentuk struktur data yang dipakai
- Estimasi time complexity
- Solusi atau jawaban ringkas
- Tautan latihan competitive programming yang serupa
- Tampilan web native dengan HTML, CSS, dan JavaScript

## Sumber Data

- OSNK2025.pdf
- OSNK2024.pdf
- OSNK2023.pdf
- OSNK2022.pdf
- KSNK2021.pdf

## Cara Pakai

Buka `index.html` langsung di browser.


## Catatan

- Analisis hanya ditujukan untuk soal yang memiliki bagian algoritmika atau source code C++.
- Untuk soal yang sangat sederhana, nama algoritma atau struktur data mungkin tidak dicantumkan bila tidak relevan.
